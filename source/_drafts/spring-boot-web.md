## RESTful web service

```
GET "localhost:8080/hello"
```

* @RestController 包含 (@Controller，@ResponseBody)
* 请求方式
    @RequestMapping(method="GET/POST/PUT/DELETE/...")
    - @GetMapping
    - @POSTMapping
    - ...

* 请求参数
    @RequestParam(value = "name", defaultValue="World")
